start_time


the beginning datetime for the event in milliseconds since the epoch
Read: anyone can read this location
Write: the event host can write to this location
This field requires that that the value is a number representing the time in milliseconds since the epoch, that the time is later than the current time, and if there is an end time already set, it must be earlier than that time
